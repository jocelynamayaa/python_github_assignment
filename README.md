# Savings Tracker 

# Description 
The Savings Tracker is a simple program that helps you estimate your yearly savings based on how much money you save each month. It prompts the user to enter their monthly savings, calculates the total amount you could save in a year, and displays the result. This program is useful for budgeting and tracking financial goals.

# How to run
Run the program using Python: 
python app.py
# or 
python3 app.py
